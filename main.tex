\section{To 剪辑工程师}

\subsection{视频类型}

视频总时长约为 03:35.

本视频使用音频流现有的歌曲音频 \textit{/mad/主音频/the.mp4}, 图像流全部使用自己的图像素材, 即 MAD. 可以参考同类型视频:

\begin{itemize}
    \item \url{https://www.bilibili.com/video/BV19E421c7gF/}
    \item \url{https://www.bilibili.com/video/BV19a4y1S7wF/}
\end{itemize}

其中, 音频流不完全复制歌曲音频: 在歌曲的伴奏部分, 对伴奏进行降低音量处理, 并增加自己的音频素材; 图像素材大部分为视频素材, 少许为图片.

\subsection{素材位置}

歌曲音频使用 \textit{/mad/主音频/the.mp4}.

增加的音频素材和视频素材混合在一起, 其中的大部分在 \textit{/mad/youtube} 文件夹中. 如使用了 \textit{/mad/bilibili独占} 或 \textit{/mad/录制} 文件夹中的视频素材会标注.

图片素材在 \textit{/mad/图片} 文件夹中.

\subsection{额外说明}

可以无视 \textit{简略}, \textit{Reference} 等章节, 直接看章节 \textit{详细}.

章节 \textit{详细} 中, \textbf{加粗的}是原音频歌词, 后面跟着图像流的内容.

为方便寻找素材内容: \textit{爱莉希雅}, 也称\textit{爱莉}, 是一个粉色头发, 使用弓箭或法杖的女性角色.
